# IsaacSim_UGV_SWARM
Swarm robotics simulation framework built with NVIDIA Isaac Sim 5.1 and ROS 2 Humble.
The system models 4 autonomous tracked UGVs equipped with LiDAR, RGB-D cameras, IMU, and GPS, enabling realistic multi-sensor navigation in simulation.
It includes EKF-based state estimation, a complete TF2 frame tree, and CycloneDDS-based communication for robust multi-robot coordination.
