# IsaacSim_UGV_SWARM
Swarm robotics simulation framework built on NVIDIA Isaac Sim 5.1  and ROS2 Humble. Features 4 autonomous tracked ground vehicles  (UGVs) equipped with LiDAR, RGB-D camera, IMU, and simulated GPS.  Includes EKF-based localization, full TF2 tree, and CycloneDDS  middleware for stable multi-robot communication.
